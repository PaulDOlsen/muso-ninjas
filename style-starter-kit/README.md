# Quasar Style Starter Kit

Copy this starter into a new app in one step:

```bash
cp -R style-starter-kit/src/css/* src/css/
```

After copying:

1. Ensure your app uses `src/css/app.scss` in Quasar config.
2. Tune brand tokens in `src/css/quasar.variables.scss` and `src/css/tokens/_colors.scss`.
3. Start composing templates with classes from `src/css/components/_patterns.scss`.

Included:

- Token files for color, spacing, radius, shadow, typography
- Base reset, typography, and layout styles
- Light and dark theme variable maps
- Reusable semantic pattern classes
- Central `app.scss` import file
